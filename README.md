# 🍼 宝宝喂奶闹钟

记录喂奶时间与奶量，准时提醒下次喂奶。

## 功能

- ⏱ **倒计时** — 实时显示距离下次喂奶的时间
- 📝 **一键记录** — 快速记录每次喂奶的奶量（ml）
- 🔔 **推送通知** — 到点提醒，支持浏览器后台推送
- 📊 **今日统计** — 喂奶次数、总奶量、平均间隔
- 📋 **历史记录** — 完整的喂奶时间线
- 🌙 **深色模式** — 自动跟随系统
- 📡 **离线可用** — Service Worker 缓存

## 部署

<https://jasonbread1198-ops.github.io/baby-feed-app>

## 使用

1. 浏览器访问上方链接
2. 点击「安装到桌面」可保存为手机 APP
3. 每次喂奶后输入奶量点击记录
4. 到时间自动提醒

## 技术栈

- 纯原生 HTML/CSS/JS
- PWA（manifest + Service Worker）
- localStorage 持久化
